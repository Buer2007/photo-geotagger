# 照片GPS轨迹合并工具

将GPS记录仪的轨迹数据（GPX文件）按时间匹配到照片的EXIF中，为照片添加地理位置信息。

## 功能特点

- 📍 **时间匹配** — 根据照片拍摄时间和GPS轨迹时间自动匹配坐标
- 🔧 **两种匹配模式** — 线性插值（精度高）和最近点匹配（速度快）
- 📁 **批量处理** — 一次性处理整个目录的照片
- 🛡️ **安全写入** — 只修改GPS标签，保留所有其他EXIF数据不变
- 🖥️ **双界面** — 命令行和图形界面两种使用方式
- 🧪 **试运行模式** — 预览匹配结果，不修改文件

## 安装

### 1. 安装Python依赖

```bash
pip install -r requirements.txt
```

依赖包：
- `gpxpy` — GPX文件解析
- `piexif-ng` — EXIF数据读写

### 2. 确认Python版本

需要 Python 3.10 或更高版本。

## 使用方法

### 命令行版 (cli.py)

**基本用法：**

```bash
python cli.py --gpx track.gpx --photos ./photos/
```

**输出到新目录（不修改原文件）：**

```bash
python cli.py --gpx track.gpx --photos ./photos/ --output ./geotagged/
```

**多个GPX文件：**

```bash
python cli.py --gpx track1.gpx track2.gpx --photos ./photos/
```

**指定时区（东八区）：**

```bash
python cli.py --gpx track.gpx --photos ./photos/ --timezone +8
```

**使用最近点匹配模式：**

```bash
python cli.py --gpx track.gpx --photos ./photos/ --mode nearest
```

**调整时间阈值（120秒）：**

```bash
python cli.py --gpx track.gpx --photos ./photos/ --threshold 120
```

**试运行（不修改文件）：**

```bash
python cli.py --gpx track.gpx --photos ./photos/ --timezone +8 --dry-run
```

### 图形界面版 (gui.py)

```bash
python gui.py
```

启动后在界面中：
1. 选择GPX轨迹文件
2. 选择照片目录
3. 设置参数（匹配模式、时间阈值、时区偏移）
4. 点击「试运行」预览结果
5. 确认无误后点击「开始处理」

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--gpx` | GPX轨迹文件路径（可多个） | 必填 |
| `--photos` | 照片目录路径 | 必填 |
| `--output` | 输出目录（留空=覆盖原文件） | 无 |
| `--mode` | 匹配模式：`interpolate` 或 `nearest` | `interpolate` |
| `--threshold` | 最大允许时间差（秒） | `60` |
| `--timezone` | 照片EXIF时间的时区偏移（小时） | `0` |
| `--dry-run` | 试运行，不修改文件 | 否 |

## 匹配模式说明

### 线性插值模式 (interpolate) — 推荐

找到照片时间戳前后的两个GPS轨迹点，在两点之间进行线性插值，计算出精确的经纬度和海拔。

**适用场景：** GPS记录仪间隔较短（1-10秒），轨迹点较密时精度最高。

### 最近点匹配模式 (nearest)

直接使用时间最接近的GPS轨迹点的坐标。

**适用场景：** GPS记录仪间隔较长，或需要快速处理大量照片时。

## 时区说明

照片EXIF中的拍摄时间通常是**本地时间**（没有时区信息），而GPX轨迹时间通常是**UTC时间**。

- 如果你在东八区（中国）拍摄，照片时间是UTC+8，需要设置 `--timezone +8`
- 如果照片EXIF包含时区信息（较少见），设置 `--timezone 0`

**建议：** 先用 `--dry-run` 试运行，检查时间差是否合理。如果时间差很大（几小时），说明时区设置有误。

## 注意事项

1. **备份照片** — 虽然工具只修改GPS标签，但建议处理前备份原始照片
2. **时间同步** — 确保GPS记录仪和相机的时间都已正确校准
3. **支持格式** — 目前支持JPEG格式的照片
4. **GPX格式** — 支持GPX 1.0和1.1标准格式，包含轨迹(tracks)和路线(routes)
